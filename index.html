FlatBum

Open Source Gallery CMS

Flatbum is an Open Source CMS which makes it simpler and easier for you to create your own self hosted Albums & Galleries with few clicks, And no database is needed.

FlatBum uses a unique algorithm to find or list any content based on date, type, category, tag, or author, and the performance will remain fast even if we have thousands of posts and hundreds of tags.

As a databaseless gallery cms, FlatBum designed to run smoothly despite using minimal server specs. With 400kb of RAM or even shared hosting, it can handle thousands of images and files without any issues.

Features

Admin Panel

Manage your galleries with ease using the built-in admin panel

Markdown

Markdown editor with preview, image upload, and draft

Multi Album

Grouping your files using albums

Support Many File Type's

Audio (mp3, wav...), Video (avi, mp4, 3gp...) And so on...

SEO Friendly

Fast loading, Clean URLs, meta canonical and description

Multi Author's

FlatBum gallery cms is multi user, And support login and signing for new users

Comments

FlatBum is ready with built-in image commenting system and with secure random words captcha

Widget

Latest posts, popular posts, related posts, archive, tag cloud

Online Backup

Download your posts and uploaded images as a Zip file

File Caching

FlatBum use aggressive file caching for performance boost

Cloud Update

No more FTP! update your HTMLy installation on the fly

Web Installer

Via Softaculous or download, run the installer, and you're done!

Databaseless

FlatBum is a flat file gallery cms, which means no database is needed

Importer

Import posts from RSS feed 2.0 into your HTMLy blog

Security

Built-in captcha to secure comments

And more...

Logs, Custom Theme Support, Analytics, Responsive design
