FlatBum

A simple databaseless flat file gallery cms

FlatBum is an open source Databaseless Gallery CMS. A simple databaseless flat file cms with simplicity and speed.

FlatBum uses a unique algorithm to find or list any content based on date, type, category, tag, or author, and the performance will remain fast even if we have thousands of posts and hundreds of tags.

As a simple php gallery, FlatBum designed to run smoothly despite using minimal server specs. With 400kb of RAM or even shared hosting, it should can handle more than 1000K galleries without any issues.

Features

Admin Panel

Manage your site with ease using the built-in admin panel

Markdown

Markdown editor with preview, image upload, and draft

Taxonomy

Grouping the contents using categories and tags

Content Type

Blog post, static page, image, video, audio, link, and quote

SEO Friendly

Fast loading, Clean URLs, meta canonical and description

Multi Author

FlatBum support user role: choose as admin or regular user

Comments

With support to built-in comment system

Widget

Latest posts, popular posts, related posts, archive, tag cloud

Online Backup

Download your posts and uploaded images as a Zip file

File Caching

FlatBum use aggressive file caching for performance boost

Cloud Update

No more FTP! update your HTMLy installation on the fly

Web Installer

Via Softaculous or download, run the installer, and you're done!

Databaseless

FlatBum is a flat file gallery cms, no database needed

Importer

Import posts from RSS feed 2.0 into your HTMLy blog

Security

Recaptcha and cryptographic hash to secure the backend

And more...

RSS feed, Sitemap.xml, Analytics, Responsive design
